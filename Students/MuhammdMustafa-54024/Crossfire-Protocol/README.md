## Project Summary
Crossfire Protocol is a browser-based, Valorant-inspired 2D tactical shooter prototype. The project focuses on small-team real-time multiplayer combat, a buy phase, plant/defuse objective flow, and server-authoritative networking built with web technologies.

## Submission Structure
This project folder contains:
- `README.md` - project overview for course submission
- `TOPIC.md` - refined topic and project scope
- `REQUIREMENTS.md` - functional and non-functional requirements
- `SKILLS.md` - required skills and learning focus
- `ROADMAP.md` - 4-week solo development roadmap
- `client/` - browser game client code
- `server/` - multiplayer server code
- `shared/` - shared types and game contracts
- `assets/` - placeholder art and media files
- `docs/` - architecture and implementation notes

## Intended Prototype Scope
- 2v2 or 3v3 tactical matches
- Browser client using HTML5 and TypeScript/JavaScript
- Node.js WebSocket game server
- Basic persistence for match stats and leaderboard
