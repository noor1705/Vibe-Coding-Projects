# Required Skills

## 1) Programming Foundations
- Strong JavaScript or TypeScript proficiency
- Asynchronous programming (Promises, event loops, concurrency patterns)
- Event-driven architecture design

## 2) Game Development Essentials
- Fixed timestep game loops
- Entity-component or entity-system modeling
- Sprite animation and camera systems
- Collision detection and resolution
- Gameplay balancing basics (damage, fire rates, recoil, economy)

## 3) Graphics and Rendering
- Canvas 2D rendering pipeline
- WebGL rendering pipeline (buffers, shaders, draw calls)
- GLSL shader fundamentals
- UI readability, hit feedback, and simple VFX for tactical combat

## 4) Multiplayer Networking
- WebSocket communication patterns
- Client-server architecture and authoritative simulation
- State synchronization (snapshots, deltas, interpolation)
- Client prediction and server reconciliation
- Latency, jitter, and packet-loss mitigation techniques

## 5) Backend and Infrastructure
- Node.js server architecture for real-time systems
- Matchmaking and round-session flow design
- Horizontal scaling, load balancing, and stateless service patterns
- Redis usage for queue/session management
- PostgreSQL schema design and query optimization

## 6) Security and Anti-Cheat
- Input validation and server trust boundaries
- Exploit detection heuristics (speed, position, action frequency)
- Secure authentication and session management

## 7) Mathematics for Game Logic
- Vectors (movement, aiming, velocity)
- Trigonometry (angles, direction vectors)
- Matrix basics for transforms (if using WebGL abstractions)
- Geometric collision math (AABB, circles, line intersections, hitscan checks)

## 8) Solo Project Focus Areas
- Gameplay engineering (movement, shooting, round flow, objective logic)
- Networking (sync, prediction, server authority)
- Backend systems (auth, lobby, persistence)
- UI and feedback (HUD, buy phase, scoreboard)
- QA/performance testing (profiling and multiplayer playtests)

## 9) Suggested Learning Order
1. TypeScript plus async/event architecture
2. Game loop, collisions, hitscan shooting, and round systems
3. WebSocket sync and server authoritative model
4. Tactical shooter economy, objective flow, and scoring
5. Matchmaking, persistence, and scaling
6. Anti-cheat hardening and observability
